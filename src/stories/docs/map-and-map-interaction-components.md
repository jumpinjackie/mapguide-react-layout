This story group focuses on the main map viewer and map-aware UI elements that react to map state.

## What is covered
- Core map display and interaction scaffolding
- Task pane and auxiliary map controls
- Readouts and overlays driven by current map state
- Viewer options and layer interaction tooling

## Notes
These stories run in a mocked environment for predictable behavior and fast iteration. Use this group to validate layout, interaction flow, and container wiring around the map viewer.
